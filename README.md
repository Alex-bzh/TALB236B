# TALB236B

Resources for L2 TAL (NLP) at Inalco.

[![Binder](https://mybinder.org/badge_logo.svg)](https://mybinder.org/v2/gh/Alex-bzh/TALB236B/main)

## Quick install

1. Install [Pixi](https://pixi.sh/), a (modern) package management tool

2. Get a clone of this repository:

```
$ git clone https://github.com/Alex-bzh/TALB236B.git
$ cd TALB236B
```

3. Install the dependencies:

```
$ pixi install
```

4. Invoke a new shell based on the environment:

```
$ pixi shell
```
